Glossário HTML

<table>
Utilizado para criação de tabelas
<p>
Utilizado para ciração de parágrafos
<em>
Utilizado para dar ênfase/itálico em textos
<strong>
Utilizado para dar importância/negrito em textos
<br>
Utilizado para quebrar linhas de texto
<pre>
Utilizado para facilitar a formatação de textos dentro do próprio código HTML
